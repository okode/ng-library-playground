# Angular Library Playground

Angular Library and App Compatibility Playground.

## Angular 10 Library

### Creation

Created with:

* Angular CLI: 10.1.4
* Node: 14.13.0

```
ng new library-ng10 --createApplication=false
cd library-ng10
ng generate library library-ng10
```

### Building

```
npm ci
ng build my-library --prod
```
