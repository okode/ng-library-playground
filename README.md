# Angular Library Playground

Angular Library and App Compatibility Playground.

## Angular 10 Library

### Creation

Created with:

* Angular CLI: 10.1.4
* Node: 14.13.0

```
ng new my-library --createApplication=false
cd my-library
ng generate library my-library
```

### Building

```
npm ci
ng build my-library --prod
```
